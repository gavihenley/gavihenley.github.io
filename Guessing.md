# Random Number Guessing Game Flowchart

## Step-by-Step Explanation

1. **Game Starts**: This is where the game begins.
2. **Pick Random Number**: The program generates a random number for the player to guess.
3. **Ask Player for Guess**: The player is prompted to input a guess.
4. **Is the Input Valid?**: The program checks if the input is a valid number within the allowed range.
   - If the input is invalid, an error message is shown, and the player is asked for another guess.
5. **Is the Guess Correct?**:
   - If the guess is too low, the player is told the guess is too low and is prompted to guess again.
   - If the guess is too high, the player is told the guess is too high and is prompted to guess again.
   - If the guess is correct, a congratulatory message is shown.
6. **End Game**: After the correct guess, the game ends.
